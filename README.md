## Overview
- This is a simple yet subtly complex typing test.
- The test has multiple features ranging from different gamemodes to different difficulty levels.
- Scores are saved and there are different postgame screens depending on your typing result.
- localStorage was implemented to save users scores and prevent constant regression upon page reload.
